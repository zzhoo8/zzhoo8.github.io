Intellij Idea中运行jfinal工程

Intellij Idea中运行JFinal工程

JFinal 推荐使用 Eclipse IDE for Java EE Developers 做为开发环境。由于使用习惯，倾向于使用Idea工具进行开发。

Idea

如果使用IDE工具进行开发，建议使用Ultimate版本，下载地址https://www.jetbrains.com/idea/#chooseYourEdition

如果需要破解，依次选择菜单Help->Register->License server，填入http://www.iteblog.com/idea/key.php，可以在线注册Idea 15的，无需任何配置

JFinal

Java web的快速开发框架

直接到官网下载demo程序http://www.jfinal.com/download?file=jfinal-2.2_demo.zip，由于demo程序是基于eclipse构建的工程，需要用Intellij Idea打开并创建maven的配置文件pom.xml

<project xmlns="http://maven.apache.org/POM/4.0.0" xmlns:xsi="http://www.w3.org/2001/XMLSchema-instance"
    xsi:schemaLocation="http://maven.apache.org/POM/4.0.0 http://maven.apache.org/maven-v4_0_0.xsd">
    <modelVersion>4.0.0</modelVersion>
    <groupId>com.bluestone</groupId>
    <artifactId>demo</artifactId>
    <packaging>war</packaging>
    <version>0.0.1-SNAPSHOT</version>
    <name>jfinal Maven Webapp</name>
    <url>http://maven.apache.org</url>
        <dependencies>
            <dependency>
                <groupId>com.jfinal</groupId>
                <artifactId>jfinal</artifactId>
                <version>2.2</version>
            </dependency>
            <dependency>
                <groupId>com.jfinal</groupId>
                <artifactId>jetty-server</artifactId>
                <version>8.1.8</version>
            </dependency>
        </dependencies>
    <build>
        <finalName>demo</finalName>

        <plugins>
            <plugin>
                <artifactId>maven-compiler-plugin</artifactId>
                <version>3.1</version>
                <configuration>
                    <source>1.7</source> <!-- yours Java version -->
                    <target>1.7</target> <!-- yours Java version -->
                    <encoding>UTF-8</encoding>
                </configuration>
            </plugin>
        </plugins>
    </build>
</project>

在DemoConfig中增加入口函数

    /**
     * 建议使用 JFinal 手册推荐的方式启动项目
     * 运行此 main 方法可以启动项目，此main方法可以放置在任意的Class类定义中，不一定要放于此
     */
    public static void main(String[] args) {
        JFinal.start("WebRoot", 8080, "/", 5);
    }

jfinal的框架必须使用jetty运行，而且JFinal框架调用的jetty的函数在最新的jetty中已经改名，因此只能使用旧版本的jetty-server 8.1.8版本

直接运行Run DemoConfig.main()即可
