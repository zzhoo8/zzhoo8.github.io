升级node.js

本机的node版本比较老旧，需要升级，用以下方法升级比较推荐

➜  temp node -v
v0.12.7

先安装node的管理模块n，然后升级node的稳定版（需要sudo）

➜  temp npm install -g n
/usr/local/bin/n -> /usr/local/lib/node_modules/n/bin/n
/usr/local/lib
└── n@2.1.0 

➜  temp n stable
