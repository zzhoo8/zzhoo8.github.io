# Mac上安装Play!框架

> https://playframework.com

## 下载最小化的Activator

https://playframework.com/download

解压缩到可访问的目录中

```bash
➜  Project tree activator-1.3.10-minimal 
activator-1.3.10-minimal
├── bin
│   ├── activator
│   └── activator.bat
└── libexec
    └── activator-launch-1.3.10.jar

2 directories, 3 files
```

系统路径加入activitor的路径，并执行生效

```bash
➜  play vi ~/.bash_profile
➜  play source ~/.bash_profile 
```
